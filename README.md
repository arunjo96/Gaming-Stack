# BitSky Gaming Platform (React.js)

A responsive and interactive gaming platform landing page built with React.js, Tailwind CSS, and Framer Motion. This project showcases animations, interactive components, and modern UI/UX features like subscription forms, social media sections, and dynamic game cards.

## Features

- **Responsive Layout**: Works across mobile, tablet, and desktop devices
- **Animated Hero Banner**: Interactive character and coin animations using Framer Motion
- **Games Section**: Popular games with hover animations, likes/favorites, and play buttons
- **Contact Section**: Email subscription with form validation and animations
- **Social Media Integration**: Interactive social media icons with hover effects
- **Swiper Carousel**: Infinite scrolling welcome messages
- **Modern UI/UX**: Buttons, cards, and banners with smooth animations and gradients
- **Scroll-to-Top**: Button that appears when scrolling down the page

## Technologies Used

- **React.js** - Frontend library
- **Tailwind CSS** - Utility-first CSS framework
- **Framer Motion** - Animation library
- **Swiper.js** - Touch-enabled carousel library
- **React Icons** - Icon library
- **JavaScript (ES6+)**
- **CSS3 & HTML5**
